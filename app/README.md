# r2Callgraph4apk UI installation

Make sure you have [node]() >= v.12.16.2. For installation instructions, refer
[here]().

Ensure you have `yarn` package manager. It can be installed using

```sh
npm install -g yarn
```
## Installation

```sh
yarn
```

## Development

```sh
yarn dev
```

## Build

```sh
yarn build
```
